Edu Flex University E-Learning Website

Features OF This Website
1. Sign In And Sign Up Page (User and Admin)
2. Home Dashboard
3. Library (Short notes, past papers, Research papers, and Books)
4. Courses
5. GPA Calculator
6. Vials
   
Using Languages
* MERNS (MongoDB, Expressjs, React, Nodejs)
